## 什么是腾讯云智能体开发平台（ADP）？

腾讯云智能体开发平台（Tencent Cloud Agent Development Platform，简称 ADP）是腾讯云推出的企业级智能体开发平台，深度整合 LLM+RAG 增强检索、自动化工作流、多 Agent 协作架构等核心能力，助力企业高效搭建稳定、安全、符合业务场景的智能体应用。

### 平台核心优势

**领先的智能体对话效果**
- 内置行业实践流程和领先的 RAG 算法
- 支持导入文档/问答对，让智能体对接企业多模态知识
- 画布式灵活编排工作流，可快速接入企业系统 API

**全面的智能体开发框架**
- 提供 LLM+RAG、Multi-agent、Workflow 等多种开发框架
- 支持应用配置-调试-评测-发布一站式工具链
- 降低智能体构建门槛，提升开发效率

**丰富的开发资源库**
- 预置精选官方插件及 MCP 插件
- 支持智能文档解析、多轮改写、embedding 等能力
- 提供智能体开发所需的核心技术组件

### 主要应用场景

**智能客服**
基于腾讯云行业大模型，提供垂直领域的专业化智能客服能力，实现知识生产效率、问题覆盖率、回答准确率跨越式提升。

**企业知识库问答助手**
支持企业级知识库搜索和智能问答，对多模态知识进行高效处理、分析、检索、问答，助力企业实现知识库数智化服务升级。

**车载助手**
结合车载系统特点，提供个性化知识问答服务，支持复杂汽车操作手册快速解析及检索，实现拟人化车内人机交互。
